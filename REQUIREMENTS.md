# Overview
React-based web application for audio recording with saving on disk feature

# Requirements

## Audio Recordings
- Controls: Record, Pause, Resume, Stop
- Visual Feedback: Display real-time waveform during recording
- Duration Limit: Maximum recording time of 4 hours
- State Management: Maintain recording state across pause/resume cycles

## Export Options
- Format: Download recorded audio as MP3
- Processing: Show loading states during conversion

## Additional Features
- Mock API for transcription
- Display transcribed text
